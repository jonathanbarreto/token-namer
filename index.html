Token Naming Tool

Generate consistent token names using structured naming conventions.

Live preview
Prefix categories are optional. Click a button to add a dropdown field.
Base tokens define category, concept, and property. Check fields and enter values. Category is required for primitive tokens; all three are required for semantic tokens.
Object tokens define group (optional), component, and element. Check fields and enter values. Component and element are required.
Optional modifiers that describe variant, state, scale, or mode. Check fields and enter values. Any entered values become suffix segments in the token name.
