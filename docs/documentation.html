Documentation

Token Namer generates token names from a set of fields. The UI is section-based (no accordions).

Frameworks

Primitive: <category>/<set>/<step>/<variant?>
Semantic: <domain>/<object>/<role>/<context?>/<state?>/<emphasis?>
Component: component/<component>/<part>/<property>/<variant?>/<state?>/<context?>

All fields accept custom input; values are normalized to kebab-case.
