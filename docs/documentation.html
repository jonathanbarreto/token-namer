How to build consistent token names, plus examples.

Naming convention

Use namespace/object/base/modifier. The modifier segment is optional; in the tool’s preview, empty segments appear as _.

Segments

Each segment should be lowercase and kebab-case (letters, numbers, and hyphens).

  • namespace: category (e.g. color, space, typography).
  • object: what it applies to (e.g. text, background).
  • base: semantic intent or role (e.g. primary, default, body).
  • modifier: optional state/variation (e.g. hover, disabled).

Examples

  • color/text/primary
  • color/text/primary/hover
  • space/md/default
  • typography/font-size/body
  • motion/duration/fast
